# 🐍 Advanced Snake Game

This is a fully functional **Snake Game** built using **HTML**, **CSS**, and **JavaScript**. The project includes modern features like keyboard activation, score tracking, self-collision detection, food generation, and snake growth—providing a smooth and interactive gameplay experience inside the browser.

## 🚀 Features

- 🎮 **Keyboard Activation Button** – Controls only work after clicking the "Keys" button.
- **Cursor Activation Button** - Controls snake's movement with cursor.
- 🍎 **Random Food Generation** – Food appears at random locations inside the game boundary.
- 🐍 **Snake Growth** – The snake grows longer when it eats food.
- 💥 **Game Over Conditions**
  - Hitting walls
  - Hitting itself (self-collision)
  - (Exception) - if playing with mouse it can hit itself.
- ⏸️ **Pause and Restart** options
- 📊 **Score Display** – Real-time score updates as snake eats food.
- ⚡ **Smooth Movement & Boundaries** – Prevents the snake from going out of bounds.

## 🧱 Technologies Used

- **HTML** – Structure
- **CSS** – Styling and layout
- **JavaScript** – Game logic and interactivity

## 🕹️ How to Play

1. Click the **Keys** or **Mouse** button to enable controls.
2. Use the **Arrow Keys** to move the snake:
   - ↑ Up
   - ↓ Down
   - ← Left
   - → Right
3. Use cursor to move the snake.
4. Eat the black food square to grow and earn points.
5. Avoid walls and yourself. If you collide, it's game over.
6. Use **Pause** to stop the game and **Restart** to reset.
