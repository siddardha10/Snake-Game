Rules

1. Select the option to play the game (Keys or Mouse)

2. Use the arrow keys to move the snake (Keys)

3. Use the mouse to move the snake (Mouse)

4. The snake will grow as it eats the food

5. If you select the mouse option, the snake can hit itself

6. The game will end if the snake hits the boundary

7. The game will end if the snake hits itself

8. The game will end if the snake hits the wall
